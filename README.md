# World Cup 2026 Predictor

Track score predictions for World Cup 2026 fixtures in a responsive, single-page web app.

## Open the app

**[Launch the World Cup 2026 Predictor](https://gavinmulvany.github.io/world-cup-predictor/)**

## Features

- Suggested scorelines informed by previous match results and xG
- Manual score entry with lock and edit controls
- Matchday progress tracking
- Copyable prediction summary
- Responsive light and dark themes

Predictions are stored in memory and reset when the page is refreshed.
